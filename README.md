# Streamlit Demo

This repository contains a small Streamlit demo app.

## Run locally

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

## Deploy

You can deploy this app using Streamlit Community Cloud by pointing it at this repo and setting the main file to `app.py`.
